Intro -

Yo! This is The Coder School’s beginner website project! If you have never made a website don’t worry this is going to be a fun learning experience and if you have some experience making a website then that’s perfectly good too. We’re going to build a webpage that is about you and your interests, kind of like a social media page but one you make yourself.

For this project, we’re going to use the repl platform. Go ahead and look around on the repl page, you should see an HTML file, a CSS file, and a JS file. These stand for HyperText Markup Language (fancy words for the text on the screen), Cascading Style Sheets (fancy words for the colors and appearance of the webpage), and JavaScript (a language that adds functions to a website). For today, we are going to only focus on HTML and CSS!
