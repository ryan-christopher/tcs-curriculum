Sprites & Backdrop //Desiging our pong game -

Sprites are the images on a Scratch computer program screen. Every Scratch program is made up of sprites and the scripts (instructions) that control them. We are able to create/get existing sprites from scratch and customize them.


Here is where we can add more sprite to our project and even design our own!


A backdrop is an image that can be shown on the Stage. It is similar to a costume, except that it is shown on the stage instead. We can design our backdrop or get pre-existing backdrops from scratch. So we need to create a tall-slim rectangle for our paddles. We would also add a small circle dot.
